# 陶渊明的五斗米

这是五斗米系列的第一篇文章，我想说说关于五斗米的事情，就是陶渊明的五斗米。在我看来，陶渊明这一生过得非常成功，从世俗的角度，陶渊明先后娶了3个老婆，生了5个孩子，这放在任何时代都不是穷人的活法。我始终认为，一个人能看不起五斗米，很大的概率是这个人有五百斗米，风骨这种东西都还是其次的。

很多地方都能查到相关的资料，陶渊明确实有5个孩子，是先后3个老婆生的。甚至还有专门的文献研究这种东西。

{% pdf 陶渊明五子年龄及诗人早年仕宦探隐.pdf %}



就算不去做官，5个孩子是不是要吃要喝，是不是需要银子去照顾衣食住行？

退一万步，当时生了5个孩子是因为那个年代的避孕手段不行，甚至没有什么避孕手段，那娶老婆总是要钱的吧？理论上陶渊明是不是有亲生父母？是不是要赡养？生病了是不是需要拿钱治病？风骨是万万填不饱肚子的。

陶渊明断断不能算是一个穷人的，具体细节大家自己去了解，建议从陶渊明那个年代的受教育比例出发。

不过，值得强调的是，他如果摧眉折腰，就是可以直接获得更好的物质生活，但是他没有这么做，不高兴挣那个钱。在他眼里，有比金钱更重要的东西。

当然，我不是说陶渊明的生活条件能好到哪里去，一个人能获得的生活水平不仅要看个体本身的财富，还要看社会的发展水平和科技的进展。就算是皇帝，生病了，局限于当时的医疗水平，还是会很遭罪。不生病的话，也没有机会周游世界的，还有非常多的掣肘。

# 2022 年居民收入

据[国家统计局](http://www.stats.gov.cn/xxgk/sjfb/zxfb2020/202301/t20230117_1892129.html)发布的数据，2022 年，全国居民人均可支配收入中位数 31370 元。

![image-20230423023430844](To-run-or-not%EF%BD%9C%E6%B6%A6/image-20230423023430844.png)

首先，有收入的人，才会有可支配收入，孩子肯定是不算在里面的。

![image-20230423023547436](To-run-or-not%EF%BD%9C%E6%B6%A6/image-20230423023547436.png)

好，现在算的是有收入的人的全国人均可支配收入的中位数是31370 元；2022 年，全国居民人均可支配收入 36883 元。中位数是平均数的85.1%。

如果现在有5个人，他们的收入分别是1，2，2，2，11。那么这5个人的平均是3.6，中位数就是2，中位数是平均数的55.6%。这个意思就是，只要你到达了平均数，那么你就超过了大多数人。你达到了中位数，你就超过了一半的人。

现在来看平均数，36883 元，每个月36883 /12=3073.58元，这是实际到手的可以自由支配的钱。如果仅考虑工资作为收入来源，那么税前的工资大概在3700的样子。

**这是什么概念，就是你只要税前工资3700元/月，你就超过了中国大多数人。**

大家要知道，中国的个税起征点是5000。具体有多少人在缴纳个人所得税，我没查到。但是从全国范围来看，大多数人的薪资都没有超过5000。

# 全国居民按收入五等份分组的人均可支配收入

![image-20230423030932472](To-run-or-not%EF%BD%9C%E6%B6%A6/image-20230423030932472.png)

![image-20230423030431450](To-run-or-not%EF%BD%9C%E6%B6%A6/image-20230423030431450.png)

这里的人均可支配收入是2021年可支配收入。注意，这里是以家庭为单位的，是要计算孩子的。2022 年，全国居民人均可支配收入 36883 元，这里算的是有收入的人，2021年是35128.1元，差不多。假设你的收入是3.5万(税后)，你的爱人的收入也是3.5万(税后)，那么你们加起来就是7万，如果你们有一个孩子，还在上学，你们的家庭人均可支配收入就是7/3=2.34万。

如果家里还有老人(没有退休金那种，也没有任何其他收入)，那就是比2.4万更低。对应的生活水平就是中等偏下，甚至是低收入组家庭。

但是，前面我们计算过，要年收入达到税后3.5万，大概就是每个月税前3700元的样子。而且这个工资是超过了中位数的，也就是超过了全国大多数人。那么现在问题来了，你和你爱人的工资超过了全国大多数人，你们整个家庭却是一个中等偏下的生活水平，甚至有可能更糟糕。

# 一万五是绝对的高薪

假设家里有一个小孩，父母两人在工作，8.5*3=25.5万元的年收入，这是实际的可支配的收入。25.5/12=2.125万元，税前大概在3万人民币，那就是1.5万的税前月薪。

这还是在数据存在极端值的情况下，马云也在高收入这个区间。假设数据是1，2，3，4，5，6，7，8，9，1001。分5等份，最后就是9和1001取平均，就是505。

注意，月薪税前1.5万，能达到高收入组的平均，是绝对的高薪。中国的贫富差距，我觉得相当大。

![image-20230423040354909](To-run-or-not%EF%BD%9C%E6%B6%A6/image-20230423040354909.png)

如果真的要以现在的眼光来看待陶渊明的话，我觉得陶渊明绝对不能算一个穷人的。他不仅先后娶了3个老婆，还生了5个孩子，在古代肯定也是有养不起孩子的，就要给别人送养，但是他都是自己养的，这说明他养得起。

# 润的成本

1. 假设润到小语种国家，你需要学语言才能融入当地社会，并且是非常熟练的程度，英语这肯定是必不可少的，但是当地的小语种也是要学的，在这种情况下，还要学习自己专业内的东西，你需要多久的时间赶上一个local的学计算机的生活水平？
2. 假设你适应环境了(这已经是非常好的情况了)，父母的养老问题怎么办呢？父母学一门新的语言，到一个新的社会，这就很困难了，对吗？
3. 你的小孩，从小接受当地的文化和当地的教育，你们就是不一样的人，和小孩会有代沟，这个概率是很大的，要怎么预防或者解决呢？

好，就算这些问题都可以解决，你能到手什么样子的生活水平？

1. 国外是美食荒漠，出门下馆子是很奢侈的事情，点外卖更奢侈，且没有时效保证，甚至，你想花钱都没有地方花钱，食物也会入乡随俗，结合当地的口味。更多的还是自己做菜，以及汉堡薯条之类的。还有可能个体的身体素质不一样，一到国外就水土不服，便秘的也是有的。
2. 看病需要预约，需要排队，可能你病都好了，预约的时间还没到。社会效率低。
3. 国外日照时间短，欧洲这边冬天的日照时间真的很短。
4. 买的衣服不合身，亚洲人的体型，和欧洲人的体型，是差别很大的。
5. 国内很多东西，比如那些义务的小商品玩意儿，在国外都很贵。

做选择没有对不对的，就看看自己能不能接受最坏的情况而已，然后再去看那个好处。

**如果以上问题都能解决，这种人真的是没什么理由在国内混不好，非要润去国外的。**有这样的解决问题的能力，在哪里都能过得很好。

# 桃花源记

以下这一段是复制粘贴的百度的《桃花源记》的翻译：

> 东晋太元年间，武陵郡有个人以打鱼为生。一天，他顺着溪水行船，忘记了路程的远近。忽然遇到一片桃花林，生长在溪水的两岸，长达几百步，中间没有别的树，花草鲜嫩美丽，落花纷纷地散在地上。渔人对此（眼前的景色）感到十分诧异，继续往前行船，想走到林子的尽头。
>
> 桃林的尽头就是溪水的发源地，于是便出现一座山，山上有个小洞口，洞里仿佛有点光亮。于是他下了船，从洞口进去了。起初洞口很狭窄，仅容一人通过。又走了几十步，突然变得开阔明亮了。（呈现在他眼前的是）一片平坦宽广的土地，一排排整齐的房舍。还有肥沃的田地、美丽的池沼，桑树竹林之类的。田间小路交错相通，鸡鸣狗叫到处可以听到。人们在田野里来来往往耕种劳作，男女的穿戴跟桃花源以外的世人完全一样。老人和小孩们个个都安适愉快，自得其乐。
>
> 村里的人看到渔人，感到非常惊讶，问他是从哪儿来的。渔人详细地做了回答。村里有人就邀请他到自己家里去（做客）。设酒杀鸡做饭来款待他。村里的人听说来了这么一个人，就都来打听消息。他们自己说他们的祖先为了躲避秦时的战乱，领着妻子儿女和乡邻来到这个与人世隔绝的地方，不再出去，因而跟外面的人断绝了来往。他们问渔人现在是什么朝代，他们竟然不知道有过汉朝，更不必说魏晋两朝了。渔人把自己知道的事一一详尽地告诉了他们，听完以后，他们都感叹惋惜。其余的人各自又把渔人请到自己家中，都拿出酒饭来款待他。渔人停留了几天，向村里人告辞离开。村里的人对他说：“我们这个地方不值得对外面的人说啊！”
>
> 渔人出来以后，找到了他的船，就顺着旧路回去，处处都做了标记。到了郡城，到太守那里拜访，报告了这番经历。太守立即派人跟着他去，寻找以前所做的标记，终于迷失了方向，再也找不到通往桃花源的路了。
>
> 南阳人刘子骥是个志向高洁的隐士，听到这件事后，高兴地计划前往。但没有实现，不久因病去世了。此后就再也没有问桃花源路的人了。

大概意思就是：

1. 吃喝不愁，每一顿都有吃的，甚至还有鸡鸭鱼肉。这就是非常好的生活了，是陶渊明对世外桃源的想象。但是现在在国内，鸡鸭鱼肉可以说是唾手可得的东西。哦对了，香菇其实是山珍海味的一种，可自行查资料。自己去菜市场买鸡鸭鱼肉，也是容易的事情。
2. 风景优美，是一座四季分明的城市，我觉得苏州可以算。
3. 桃花源里这些人家，放到现在的水平，就是这个表里面的东西都有。每一百户人家，有家用汽车的只有41.8，这还是平均数，好多家庭都不止一辆车，好多家庭一辆车也没有。甚至有的家里都没有洗衣机的。
   ![image-20230423044320710](To-run-or-not%EF%BD%9C%E6%B6%A6/image-20230423044320710.png)

其他的标准大概是，拥有自己独立的房间，或者自己能够负担得起独立的房间，买的租的都可以。工作不经常加班也是很重要的，怎么着也得双休吧。

我说这些，是想说，一个人的生活水平，是有上限的，这个限制可以是国家的政策，科技的发展水平，社会的运作效率等等。当然，个人的喜好也算在内的，比如有的人就是喜欢夜夜笙歌，那么消费自然就上去了，上不封顶。但是客观地说，吃喝不愁，拥有自己独立的房间和浴室，在一座四季分明的城市，有一份双休的工作(得交五险一金和个税的那种)，不经常加班，这是很高的生活水平，但是前提是没有其他压力，比如购房压力，比如被家人拖累，比如情感困惑之类的问题等等。

最后，其实这个世界上不存在幸福这个事情，你要让我去过皇帝的生活，我可能也不愿意的，很明显，现代的生活更好，至少对于普通人来说，是很好的。由于科技的进步和时代的发展，我们可以日行千里，可以去旅游，可以认识到非常多的东西，甚至按照以前的速度，根本就不可能接触到的东西。甚至，退一万步，人生的试错成本降低了，以前作为一个女性，需要依附家庭才能生存，但是现在可以有机会靠自己实现真正的独立，并且不需要依附谁。

**幸福都是比较出来的，你比别人好，或者你比过去的你自己要好，你就会感到幸福。我真觉得幸福这种东西，就是比较的产物。**这就是人类虚伪的地方之一。

虽然我主观上觉得自己是一个很贫困的人，但是如果客观地说，看着中国统计年鉴上的数字，其实我根本就没有见过中国大多数人。中国有14亿人口，但是月薪5000，已经是大多数人都达不到的工资水平了。我周围的本科生和研究生，5000块钱肯定还是有的。不过我还是觉得自己很贫困哈哈哈，努力学习，努力挣钱。

陶渊明，作为高级知识分子，在那个万般皆下品惟有读书高的年代，肯定不能算穷人啦。

# 写在最后

我觉得陶渊明是很伟大的人。他确实有贫困潦倒的时候，但是他没有抱怨生活，不管他是不是装的，他装了一辈子，他就是真的。官场是很复杂的，虽然我也搞不清楚为什么某些从政的人非要欺负别人，也许在他们看来那并不叫欺负。就是单纯的有那个能力，有那个合法的伤害权，简言之，就是他们手中的权力可以对你造成伤害，但是行为和操作方式都是合法的，就想用一用那个权力。

很多事情是避免不了的，一个人无法预判生命中会有的灾难和挫折，甚至，就算处处小心翼翼，也可能在不经意间功亏一篑，比如生病什么的。目前在中国，得了癌症，医保的钱也是非常有可能不够花的，甚至可以让一个家庭一贫如洗。在欧美国家，又有养老的问题，人力成本太高了。

一个人能遇到的困难真的太多了，生活本身就是很艰难的，希望大家一切都好。
